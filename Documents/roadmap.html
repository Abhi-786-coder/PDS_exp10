[!IMPORTANT] This is the authoritative, fully-updated roadmap. It supersedes the original 6-phase plan. All upgrades from the pipeline corrections, imbalance strategy, and literature review are integrated here.

What This Project Actually Is

Not a toxicity classifier. A Prediction-to-Prescription decision-support system that:

  1. Predicts toxicity across all 12 Tox21 endpoints simultaneously
  2. Identifies which molecular fragment is responsible (SHAP × structural alert validation)
  3. Suggests synthesizable bioisostere replacements (ChEMBL-backed, SAScore-filtered)
  4. Re-predicts across all 12 endpoints for each candidate and outputs a Pareto front
  5. Attaches calibrated uncertainty intervals and OOD flags to every prediction

Closest prior art: Cyto-Safe (Feitosa et al.) — which stops at step 2. You go through step 5.
Key literature: Zhang et al. (2025), Barua et al., Feitosa et al. (your extracted papers)

Final Research Question

"Does the geometric cohesion of each Tox21 endpoint's toxic class (measured by intraclass Tanimoto similarity) predict where SMOTE-augmented training and SHAP attribution systematically fail — and does a SHAP-guided, structurally-validated, synthesizability-filtered bioisostere pipeline demonstrate the practical consequence of those failures by producing Pareto-dominant reductions across all 12 endpoints for high-cohesion vs. low-cohesion scaffolds?"

SMILES Validation Gate — Run First, Always

[!WARNING] Every downstream function assumes valid SMILES. Chem.MolFromSmiles() returns None silently for invalid inputs, which propagates as IndexError or NaN loss hours into training. Run this once at ingestion and index everything downstream off the validated list. 
from rdkit import Chem
import pandas as pd

def validate_smiles(smiles_list: list) -> tuple:
    valid_indices, invalid_records = [], []
    for i, smi in enumerate(smiles_list):
        mol = Chem.MolFromSmiles(smi)
        if mol is not None:
            valid_indices.append(i)
        else:
            invalid_records.append((i, smi))
    if invalid_records:
        print(f"WARNING: {len(invalid_records)} invalid SMILES dropped:")
        for idx, smi in invalid_records[:5]:
            print(f"  Row {idx}: '{smi[:50]}'")
    print(f"✅ {len(valid_indices)}/{len(smiles_list)} valid SMILES")
    return valid_indices, invalid_records

Phase 2 — Multi-Representation Featurization + Scaffold Split

Why This Phase is Novel

Instead of assuming Morgan fingerprints (as every other project does), you run an ablation study across 5 representations. Zhang et al. (2025) Table 2 documents these theoretically — you are the first to compare them empirically on Tox21 using AUPRC as the primary metric and SHAP compatibility as an evaluation criterion.

Morgan Fingerprints WITH bitInfo Capture

[!WARNING] Passing bitInfo={} inside the function call creates a temporary dict that gets garbage collected immediately. This causes SHAP-to-atom mapping in Phase 6 to fail silently. Declare the dict OUTSIDE the function call. 
from rdkit import Chem
from rdkit.Chem import AllChem
import numpy as np

def smiles_to_morgan_with_info(smiles: str, radius: int = 2, n_bits: int = 2048) -> tuple:
    """Returns (fingerprint_array, bit_info_dict)."""
    mol = Chem.MolFromSmiles(smiles)
    if mol is None:
        return None, None
    
    bit_info = {}  # ✔ Declared OUTSIDE
    fp = AllChem.GetMorganFingerprintAsBitVect(
        mol, radius=radius, nBits=n_bits, bitInfo=bit_info
    )
    return np.array(fp), bit_info

def smiles_to_morgan(smiles: str, radius: int = 2, n_bits: int = 2048) -> np.ndarray:
    """Fingerprint only, no bit info (for fast training loops)."""
    fp_array, _ = smiles_to_morgan_with_info(smiles, radius, n_bits)
    return fp_array

If you feed unscaled combinations into PCA (Phase 3) or ToxNet (Phase 4), the large-magnitude variables will completely destroy the gradients and PCA variance calculations. You MUST scale continuous descriptors.

Scaffold Splitting (Mandatory — Not Optional)

[!WARNING] Random splitting on molecular data causes structural leakage — similar scaffolds appear in both train and test, inflating AUC falsely. Use Murcko scaffold split. Zhang et al. (2025) and Barua et al. both use scaffold splits for honest evaluation. 
from rdkit.Chem.Scaffolds import MurckoScaffold
from collections import defaultdict

def scaffold_stratified_split(smiles_list, labels, test_size=0.2, val_size=0.1):
    scaffold_to_idx = defaultdict(list)
    for i, smi in enumerate(smiles_list):
        mol = Chem.MolFromSmiles(smi)
        if mol:
            scaffold = MurckoScaffold.MurckoScaffoldSmiles(mol=mol)
            scaffold_to_idx[scaffold].append(i)
    
    scaffolds = list(scaffold_to_idx.keys())
    np.random.shuffle(scaffolds)
    
    n_test = int(len(smiles_list) * test_size)
    n_val = int(len(smiles_list) * val_size)
    test_idx, val_idx, train_idx = [], [], []
    
    for scaffold in scaffolds:
        indices = scaffold_to_idx[scaffold]
        if len(test_idx) < n_test: test_idx.extend(indices)
        elif len(val_idx) < n_val:  val_idx.extend(indices)
        else:                        train_idx.extend(indices)
    
    return np.array(train_idx), np.array(val_idx), np.array(test_idx)

Phase 3 — Geometric Imbalance Analysis + Focal Loss Training

Step 3A — Geometric Imbalance (Novel Research Finding)

Before applying any resampling, measure whether the toxic class is geometrically clustered. This predicts where SMOTE is valid and where it isn't — a finding no prior Tox21 paper has quantified.

from rdkit import DataStructs

def compute_intraclass_cohesion(smiles_list, labels, target_name, sample_size=200):
    fps = [AllChem.GetMorganFingerprintAsBitVect(Chem.MolFromSmiles(s), 2, 2048)
           for s in smiles_list if Chem.MolFromSmiles(s)]
    
    toxic_idx   = np.where(labels == 1)[0]
    nontoxic_idx = np.where(labels == 0)[0]
    
    def mean_pairwise(idx):
        if len(idx) > sample_size:
            idx = np.random.choice(idx, sample_size, replace=False)
        fps_sub = [fps[i] for i in idx]
        sims = [DataStructs.TanimotoSimilarity(fps_sub[i], fps_sub[j])
                for i in range(len(fps_sub)) for j in range(i+1, len(fps_sub))]
        return np.mean(sims) if sims else 0.0
    
    toxic_coh    = mean_pairwise(toxic_idx)
    nontoxic_coh = mean_pairwise(nontoxic_idx)
    ratio        = toxic_coh / max(nontoxic_coh, 1e-6)
    
    return {
        'endpoint': target_name,
        'toxic_cohesion': round(toxic_coh, 4),
        'nontoxic_cohesion': round(nontoxic_coh, 4),
        'cohesion_ratio': round(ratio, 2),
        'smote_valid': ratio < 1.5,   # SMOTE valid only if NOT highly clustered
    }

Step 3B — Focal Loss (Primary Loss Function)

[!IMPORTANT] Do NOT use standard BCE or class_weight='balanced' as primary. Focal Loss (γ=2) with per-endpoint alpha weights is the correct approach. Barua et al. validate SMOTE + hyperparameter optimization as the field-standard combination — Focal Loss is the stronger alternative to SMOTE for severely imbalanced endpoints. 
import torch
import torch.nn.functional as F

class PerEndpointFocalLoss(torch.nn.Module):
    def __init__(self, pos_weights: dict, target_names: list, gamma: float = 2.0):
        super().__init__()
        self.gamma = gamma
        raw = torch.tensor([pos_weights[t] for t in target_names])
        self.alpha = (raw / (raw + 1)).unsqueeze(0)  # Per-endpoint alpha: (1, 12)
    
    def forward(self, logits, targets):
        # targets: -1 for unknown labels → mask those out
        mask           = (targets != -1).float()
        targets_clean  = targets.clone().float()
        targets_clean[targets == -1] = 0.0
        
        alpha = self.alpha.to(logits.device).expand_as(logits)
        probs = torch.sigmoid(logits)
        p_t   = probs * targets_clean + (1 - probs) * (1 - targets_clean)
        alpha_t = alpha * targets_clean + (1 - alpha) * (1 - targets_clean)
        
        bce   = F.binary_cross_entropy_with_logits(logits, targets_clean, reduction='none')
        focal = alpha_t * (1 - p_t) ** self.gamma * bce
        
        masked = focal * mask
        return masked.sum() / mask.sum().clamp(min=1)

Step 3C — SMOTE in Embedding Space (Only Where Valid)

# Only for endpoints where geometric_analysis['smote_valid'] == True
from imblearn.over_sampling import ADASYN
from imblearn.under_sampling import TomekLinks

def resample_if_valid(embeddings, y_endpoint, smote_valid):
    if not smote_valid:
        return embeddings, y_endpoint    # Don't SMOTE geometric clusters
    
    adasyn = ADASYN(random_state=42, n_neighbors=5)
    X_res, y_res = adasyn.fit_resample(embeddings, y_endpoint)
    
    tomek = TomekLinks()
    X_clean, y_clean = tomek.fit_resample(X_res, y_res)
    return X_clean, y_clean

Phase 4 — ToxNet Training with OPTUNA

Architecture: Shared Backbone + Per-Endpoint Heads

import torch.nn as nn

class ToxNet(nn.Module):
    def __init__(self, input_dim=2048, shared_dims=[1024,512,256], head_dim=64, n_tasks=12):
        super().__init__()
        layers, prev = [], input_dim
        for h in shared_dims:
            layers += [nn.Linear(prev,h), nn.BatchNorm1d(h), nn.GELU(), nn.Dropout(0.3)]
            prev = h
        self.backbone = nn.Sequential(*layers)
        
        # 12 independent heads — one per Tox21 endpoint
        self.heads = nn.ModuleList([
            nn.Sequential(nn.Linear(prev,head_dim), nn.ReLU(), nn.Dropout(0.15), nn.Linear(head_dim,1))
            for _ in range(n_tasks)
        ])
    
    def forward(self, x):
        shared = self.backbone(x)
        return torch.cat([h(shared) for h in self.heads], dim=1)
    
    def get_embeddings(self, x):
        return self.backbone(x)

Phase 5 — Evaluation Suite

Primary Metric: AUPRC (Not AUROC, Never Accuracy)

[!CAUTION] The original roadmap listed ROC-AUC as the primary metric. This is wrong for severe imbalance. AUPRC is more informative when the minority class is < 10%. A random classifier gets AUPRC = prevalence (e.g., 0.05 for NR-AR-LBD). Every result must be reported with AUPRC Lift = AUPRC / prevalence to show improvement over random. 
from sklearn.metrics import roc_auc_score, average_precision_score, f1_score, matthews_corrcoef

def evaluate_all_endpoints(Y_true, Y_proba, Y_pred, TARGET_COLS):
    results = []
    for i, target in enumerate(TARGET_COLS):
        mask = Y_true[:, i] != -1
        y_t, y_p, y_d = Y_true[mask, i], Y_proba[mask, i], Y_pred[mask, i]
        
        prev   = y_t.mean()
        auprc  = average_precision_score(y_t, y_p)
        auroc  = roc_auc_score(y_t, y_p)
        f1     = f1_score(y_t, y_d, zero_division=0)
        mcc    = matthews_corrcoef(y_t, y_d)
        
        results.append({
            'Endpoint': target,
            'Prevalence': f"{prev:.1%}",
            'AUPRC': round(auprc, 4),           # ← PRIMARY
            'AUPRC_Lift': f"{auprc/prev:.1f}x", # vs random
            'AUROC': round(auroc, 4),
            'F1': round(f1, 4),
            'MCC': round(mcc, 4),
        })
    return pd.DataFrame(results)

Per-Endpoint Threshold Tuning (Recall Floor = 0.85)

from sklearn.metrics import precision_recall_curve

def tune_threshold(y_true, y_prob, recall_floor=0.85):
    precs, recs, thresholds = precision_recall_curve(y_true, y_prob)
    thresholds = np.append(thresholds, 1.0)
    valid = recs >= recall_floor
    best_idx = np.argmax(precs * valid) if valid.any() else np.argmax(recs)
    return {
        'threshold': round(thresholds[best_idx], 4),
        'recall':    round(recs[best_idx], 4),
        'precision': round(precs[best_idx], 4),
    }

Phase 6 — Note on ChEMBL API

[!WARNING] The prescription pipeline previously queried ChEMBL live in the Streamlit dashboard. ChEMBL enforces rate limiting and common fragments return hundreds of results. This causes the dashboard to hang or crash. The correct approach is a one-time cache build run as an offline script, saved as a local database or pickle, and queried locally at runtime. 
# OFFLINE SCRIPT (run once, not in dashboard):
# build_chembl_cache.py

from chembl_webresource_client.new_client import new_client
import pickle

FRAGMENT_QUERIES = [
    'Nc1ccccc1',     # Aniline
    '[NX3](=O)=O',   # Nitro
    'c1ccc(F)cc1',   # Fluorobenzene
    'c1ccc(O)cc1',   # Phenol
]

cache = {}
for frag in FRAGMENT_QUERIES:
    molecule = new_client.molecule
    results = molecule.filter(
        molecule_structures__canonical_smiles__contains=frag
    ).only(['molecule_chembl_id', 'molecule_structures', 'molecule_properties'])

The 4-Step Corrected Flow

Step 1: Predict all 12 endpoints → SHAP per endpoint
Step 1b: Cross-validate SHAP against Brenk + PAINS + OCHEM alerts
         → Flag low-confidence attributions (Zhang et al. open problem)
         → Only proceed if alert overlap confirmed

Step 2: Map top SHAP bit → molecular fragment (RDKit atom-info)

Step 3: Query ChEMBL API for compounds containing replacement fragment
         → Filter A (Synthesizability): SAScore < 4.0
         → Filter B (Pharmacokinetics): |ΔLogP| < 0.5 and |ΔMW| < 25 Da 
           (Prevents bioisosteres that destroy drug absorption/efficacy)
         → Flag: Tanimoto to DrugBank < 0.4 (out-of-pharma-space warning)

Step 4: Re-predict all 12 endpoints for each viable candidate
         → Temperature-scaled calibrated probabilities
         → Apply "Intended Mechanism" override (ignore flags if target is intentional)
         → OOD detection via Tanimoto to training set
         → Pareto dominance classification
         → Output: sorted Pareto front with ADME metrics and uncertainty intervals
